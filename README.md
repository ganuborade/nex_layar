# NexLayar

NexLayar is a clean, responsive business website built for web development services.

## Overview

This project is a simple static website showcasing:
- Web development services
- Portfolio projects
- Pricing plans
- Contact section using Google Forms

## Files

- `index.html` — main website page
- `src/styles/style.css` — styling for the site
- `src/scripts/main.js` — interactive frontend behavior
- `package.json` — project metadata
- `LICENSE` — license file
- `public/images/favicon.ico` — favicon asset

## Contact

The contact section uses a Google Form embed instead of a local form or backend.

## How to use

1. Open `index.html` in your browser.
2. The site is fully responsive and works without a backend.

## Notes

- This is a minimal static website, ready to deploy on GitHub Pages, Netlify, or any static host.
